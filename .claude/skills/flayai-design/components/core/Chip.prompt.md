Fully-rounded pill. `role="suggestion"` for first-screen example queries (card bg, hover tints); `role="option"` for compact search-option pills (개수 / 종류). `selected` paints the Apple-blue active state.

```jsx
<Chip onClick={() => send(q)}>온천에서 즐기는 영상</Chip>
<Chip role="option">10</Chip>
<Chip role="option" selected>Instance</Chip>
```
